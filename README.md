# 🌐 Visor 360 Web

Aplicación web para visualizar imágenes panorámicas 360° (equirectangulares)

## 🚀 Demo en vivo

👉 https://anibal0645.github.io/visor-360-web/

## 🧩 Características

* Visualización de imágenes 360°
* Carga de imágenes locales
* Soporte para imágenes por URL
* Navegación interactiva (zoom y rotación)

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Pannellum (visor 360)

## 📷 Requisitos

* Imagen panorámica equirectangular (relación 2:1)

## 📦 Uso

1. Abre la página web
2. Carga una imagen local o ingresa una URL
3. Interactúa con la vista 360°
